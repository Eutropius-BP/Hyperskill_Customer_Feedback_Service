# Customer Feedback Service

A simple CRUD web service built with Spring Boot and MongoDB to store and retrieve customer feedback.

## Tech Stack
- Java 17
- Spring Boot
- MongoDB
- REST API

## Features
- Create, read, update, delete feedback entries
- Repository pattern using Spring Data MongoDB
- API endpoints for searching and querying feedback
- Data validation & error handling
